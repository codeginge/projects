# projects

When adding a project, follow this organizational structure:
```
/project_name
  ├── README.md
  ├── design/                 
  │     ├── part1.stl
  │     ├── part2.stl
  ├── docs/
  │     ├── instructions.pdf
  │     ├── overview.md
  └── other/
        ├── data.csv
        ├── design.png
```
