AUTO CODER 

The Auto Coder will convert written code to arduino c++ and then using functional tests auto grade the code. Any errors will be captured and returned to the student. 

1. coding flow
    1. Capture camera - IPEVO Document camera
    1. Open CV image to black and white
    1. Send through qwen2.5-VL to extract text
    1. compile and upload to arduino
    1. run functional testing and return grade
